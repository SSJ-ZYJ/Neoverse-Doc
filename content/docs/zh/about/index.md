本项目旨在构建一个具有独特科技美学风格的文档站点模板，采用 Next.js 16 和 React 19 构建，实现纯静态化导出，完美适配边缘节点托管。

## 功能亮点

- 高性能静态生成 (SSG)
- 液态玻璃质感与动态高光主题
- 多语言 i18n 支持
- 全终端沉浸式阅读体验
