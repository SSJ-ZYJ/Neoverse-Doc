This project aims to build a documentation site template with a distinctive tech aesthetic, powered by Next.js 16 and React 19, with pure static export for seamless edge-node hosting.

## Highlights

- High-performance static generation (SSG)
- Liquid glass surfaces and dynamic-highlight theming
- Full multilingual i18n support
- Immersive reading experience across all devices
